Документация по тестированию приложения "ИМТ"

Требования:

	Пк
    	Программа pyton
    	Визуализатор кода

Объекты тестирования: Файл imt.py

Не нужно тестировать файл Readme.md

По запуску кода на исполнение: Динамическое тестирование

По доступу к коду и архитектуре: Тестирование белого ящика

По уровню детализации приложения: Приемочное тестирование

По степени автоматизации: Ручное тестирвоание

По уровню функционального тестирования: Расширенное тестирование

В зависимости от исполнителей:э Альфа-тестирование

Если приложение запускается и весь функционал приложения работает, то тестирование прошло успешно

Если приложение ни как не запускается, необходимо связаться с разработчиком, который написал код

Результаты проведения тестирования:

    	Приложение запускается
	Возможность внесения показателей в поля
    	При нажатии на кнопку, вылезает ответ с результатом
    	Над числами проводятся корректные математические операции

Задачи для проведения тестирования: Убедиться, что приложение запускается и корректно работает (выдает корректные ответы)
